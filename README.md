## 💻 Sobre StopWatchPro

"StopWatchPro.com" es el compañero perfecto para tus necesidades de cronometraje en línea. Con su diseño intuitivo y funciones prácticas, este cronómetro te permite realizar un seguimiento preciso del tiempo con facilidad. Ya sea que necesites medir el tiempo de tus ejercicios, proyectos o actividades cotidianas, StopWatchPro.com lo hace posible con la capacidad de iniciar, pausar y reiniciar el cronómetro según tus necesidades. Su interfaz clara y botones de control accesibles garantizan una experiencia de usuario sin complicaciones. ¡Potencia tu productividad y controla el tiempo con precisión utilizando StopWatchPro.com ahora mismo!"

## ✅ Requisitos

- Windows 10 o superior
- Github
- Visual Code
- Extension "Live Server" para Visual Code

> [ !IMPORTANT]
> Para que funcione el proyecto en su PC debe cumplir con el **todos** los requisitos

## 💿 Instalacion y uso

- Clonar repositorio:

```bash
git clone https://github.com/CristianDecidueye/Punto5DOM-BOM-cronometro
```

- Asegurarse que se encuentra en la rama `main`
- Ejecutar el archivo `index.html` con la extension Live Server
- Una vez iniciada la aplicacion, la misma se podra previsualizar en el siguiente link `http://127.0.0.1:5500/index.html`

## 🤝 Como contribuir?

Para contribuir, por favor seguir los siguientes pasos:

1. Hacer un Fork del repositorio y clonarlo
2. Crear desde la rama `dev` una nueva rama para comenzar `git checkout -b nueva-rama`
3. Realizar los cambios y subirlos a su fork `git push origin nueva-rama`
4. Solicitar un Pull Request a la rama `dev` al repositorio original
